Detailed Description

template<class Ctx>
class MultiWorldTesting::BootstrapExplorer< Ctx >

The bootstrap exploration class.

The Bootstrap explorer randomizes over the actions chosen by a set of default policies. This performs well statistically but can be computationally expensive.
